Проект: Выбираем авто выгодно.

Описание: Хакатон. Сореновательный проект на платформе Kaggle по продаже автомобилей с пробегом в Москве.

Задача:  создать ML модель, которая будет предсказывать стоимость автомобиля по его характеристикам.

Критерий оценки: метрика MAPE  (Mean Percentage Absolute Error - средняя абсолютная ошибка в процентах).



/README - этот файл
/Auto_select.ipynb - рабочий ноутбук, в котором выполняются все вычисления и решения
/function.py - файл с функциями



в данном проекте выполнена обработка датасета, заполнены пустые значения, выполнено обогащение дополнительными признаками, произведен разведывательный анализ.

По итогам EDA выявлены закономерности в параметрах исследумых данных, влияющие на результат предскания.

Получена метамодель LinearRegression, составленная на основе бустерной (GradientBoostingRegressor) и 
древовидной (ExtraTreesRegressor) модели.
Метамодель, обученная на исходном датасете показывает результат на учебной выборке:

MAPE = 15.38
